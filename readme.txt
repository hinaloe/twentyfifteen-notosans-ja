== Description ==

The font of TwentyFifteen is changed to a Japanese Gothic font.　　

This plugin replace TwentyFifteen's **Noto** font to *Noto Sans Japanese* web font. (which distributed in https://www.google.com/fonts/earlyaccess )

This plugin only works TwentyFifteen and it's child theme.

This plugin is published on [GitHub](https://github.com/hinaloe/twentyfifteen-notosans-ja).

----

TwentyFifteen に使用されている `Noto Sans` 及び `Noto Serif` を `Noto Sans Japanese` に置き換えます。本来セリフ・明朝の部分もゴシック体になるのでご注意下さい。
このプラグインは TwentyFifteen とその子テーマ以外では機能しません。



== Installation ==

1. Upload `twentyfifteen-notosans-ja` dir to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
